# 项目概览
carbon-ai-game -- 一个在课堂使用的连词成句AI程序

## 技术栈
- 语言：TypeScript
- 框架：Electron

## 项目结构
└─carbon-ai-game  // 连词成句游戏的主程序

## 代码风格与规范
- 类型：严格规范代码中定义的变量、结构体类型，避免强制转换、类型断言，TypeScript避免使用any类型
- 导入顺序：标准库>第三方库>自定义库
- 注释：每个函数、方法在开头处简要描述功能，关键代码部分添加注释描述具体业务功能
- 规范：代码中避免出现魔鬼数字，避免注释代码，无效代码及时删除，完成每项小任务后及时在git提交代码并推送远端服务器

## 架构与设计约束
- 安全：涉及密码的部分遵循当前主流安全设计，禁止使用DES、MD5等不安全算法。服务端采用HTTPS链接，采用自签名证书
- 设计：这是一个演示项目，只需要实现必要的业务能力
- 架构：这是一个桌面程序
