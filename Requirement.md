# 开发需求
carbon-ai-game 一个在课堂上使用的连词成句AI游戏程序

## 第一阶段需求
使用Electron框架，开发一个桌面端程序，实现连词成句的游戏。具体工作流程如下：
1. 从配置文件中读取一个词语组合，配置文件结构如下：
```yaml
- words: ["虽然", "但是"]
  used: false
- words: ["因为", "所以"]
  used: true
```
其中`words`字段中的词语表示这个题目中要求用于造句的词语，回答的句子中应该包含列出的词语；`used`字段表示这个题目是否已经使用过，`true`表示已经使用过，出题时将不再出现这个题目，当使用过这组词语后，程序应该将`used`字段修改为`true`。
2. 获取玩家造的句子，程序同时提供两种输入模式：手写模式和键盘输入模式。当玩家有手写输入时，使用硅基流动的DeepSeek-OCR模型识别成文本
3. AI点评，将连词造句的题目和玩家输入的句子提交给DeepSeek-V4-Pro大模型，从是否使用了所需的词语、语句是否有语法问题、语句是否通顺自然等维度评分并点评。将AI评价的显示在界面上
